# Frontend MCP Loader
For frontend rework/UI improvements:
`claude mcp add magic-ui --scope project -- npx -y @magicuidesign/mcp@latest`
`claude mcp add shadcn --scope project -- npx -y shadcn-ui-mcp-server`
Restart session after adding.
