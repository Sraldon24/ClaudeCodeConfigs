# Hooks
- **Types**: PreToolUse (validation), PostToolUse (format), Stop (verify)
- **Permissions**: Auto-accept for trusted plans. Disable for exploratory. No `dangerously-skip-permissions` (use `allowedTools`).
- **TodoWrite**: Track multi-step tasks, real-time steering. Reveals out-of-order, missing, extra, or misinterpreted steps.
